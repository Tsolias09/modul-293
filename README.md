# Nikos Skartsolias – Persönliches Fussball-Portfolio

## 1. Projekttitel

**Nikos Skartsolias – Persönliches Fussball-Portfolio**

Dieses Projekt ist ein digitales Portfolio über den Fussballspieler **Nikos Skartsolias**. Die Website präsentiert den Spieler, seine sportliche Entwicklung, seine Stärken, seine Spiele, Medieninhalte, Highlights, Ziele und Kontaktmöglichkeiten in einem modernen Webauftritt.

Die Website ist als persönliches Portfolio aufgebaut und soll professionell wirken, damit sie zum Beispiel von Trainern, Scouts, Vereinen oder interessierten Personen angeschaut werden kann.

---

## 2. Kurzbeschreibung

Das Portfolio stellt Nikos Skartsolias als jungen ambitionierten Fussballer vor. Der Fokus liegt auf einer modernen, hochwertigen und sportlichen Präsentation. Die Gestaltung orientiert sich an Premium-Sport-Websites, Spielerprofilen und modernen Portfolio-Websites.

Die Website enthält mehrere Hauptseiten:

- **Home** – Startseite mit Hero-Bereich und persönlichen Bereichen
- **About Me** – persönliche Geschichte, Werte und Entwicklung
- **Player Profile** – Spielerprofil mit Position, Körperdaten und Stärken
- **Career** – sportliche Entwicklung und Stationen
- **Games** – Spiele, Resultate, Rangliste und Saisonstatistik
- **Statistics** – Leistungsdaten
- **Performance** – Training, Entwicklung und Leistungsbereiche
- **Media** – Bildergalerie
- **Highlights** – besondere Momente
- **Vision** – Ziele und Zukunft
- **Contact** – Kontaktformular und Team-/Kontaktinformationen

---

## 3. Ziel des Projekts

Das Ziel war es, ein digitales Portfolio zu erstellen, das nicht wie eine einfache Schulwebsite wirkt, sondern wie eine moderne persönliche Spielerwebsite.

Die Website soll:

- Nikos als Fussballspieler professionell vorstellen
- wichtige Informationen übersichtlich darstellen
- auf Mobile, Tablet und Desktop funktionieren
- moderne Webdesign-Techniken einsetzen
- mit HTML, CSS und JavaScript ohne Framework umgesetzt sein
- durch KI-Unterstützung effizient entwickelt und verbessert werden
- als Projekt für Modul 293 sauber dokumentiert sein

## 4. Zielgruppe

Die Website richtet sich an folgende Personen:

## 8. Hauptfunktionen

### 8.1 Navigation

Die Website besitzt eine einheitliche Navigation auf allen Seiten. Auf mobilen Geräten wird sie als Hamburger-Menü dargestellt.

### 8.2 Startseite

Die Startseite enthält:

- grosse Hero-Section
- Name und Position
- Call-to-Action Buttons
- persönliche Karten
- Links zu Unterseiten

### 8.3 Persönliche Bereiche

Auf der Startseite werden persönliche Bereiche wie Geschichte, Training und Werte dargestellt. Diese Karten besitzen Hover-Effekte und führen zu passenden Bereichen der Website.

### 8.4 Games-Seite

Die Games-Seite enthält:

- letztes Spiel
- Resultat
- Spielplan
- Filterfunktion
- Rangliste
- Saisonstatistik

Die Tabelle ist horizontal scrollbar, damit sie auch auf kleineren Geräten gut funktioniert.

### 8.5 Media-Seite

Die Media-Seite zeigt Bilder in einem modernen Grid. Die Bilder sind so gestaltet, dass sie die Cards ausfüllen und visuell professionell wirken.

### 8.6 Kontaktseite

Die Kontaktseite enthält:

- Kontaktformular
- Betreff-Auswahl
- Team-/Kontaktinformationen

## 9. Responsives Design

Die Website wurde für verschiedene Bildschirmgrössen optimiert:

- Desktop
- Laptop
- Tablet
- Mobile

Wichtige responsive Anpassungen:

- Navigation wird auf Mobile zum Hamburger-Menü
- Cards werden untereinander gestapelt
- Tabellen werden horizontal scrollbar
- Bilder passen sich an die Card-Grösse an
- Schriftgrössen werden mit `clamp()` und Media Queries angepasst
- Grid-Layouts wechseln von mehreren Spalten auf eine Spalte

## 10. Besonderheiten

### Premium Dark Theme

Die Website nutzt ein dunkles Design mit Gold-Akzenten. Dadurch wirkt sie hochwertig und sportlich.

### Glassmorphism

Mehrere Cards verwenden halbtransparente Hintergründe mit weichen Borders und Schatten.

### Games-Daten

Die Games-Seite zeigt echte Wettbewerbsdaten im Stil einer professionellen Spielerwebsite.

- Name
